# 오감리뷰 광고 포털 사이트 구축

## 프로젝트 개요
오감리뷰(Oganm Review) 광고 가이드라인 포털 사이트 구축.
**대상**: 오늘의집 파트너사(광고주) — 신뢰감 있고 세련된 디자인으로 파트너사에게 광고 정보를 명확하게 전달하는 것이 목표.

---

## 디자인 요구사항
- **참고 스타일**: Toss Payments 개발자 센터 (https://docs.tosspayments.com/guides/v2/get-started)
  - 사이드바 메뉴 구조 참고
- **배경색**: `#FFFFFF`
- **포인트 컬러**: `#1496F5` (오감리뷰 상징색)
- **폰트**: `Pretendard`
- **전체 분위기**: 신뢰감을 주는 깔끔하고 세련된 디자인

---

## 기능 요구사항
1. **반응형 레이아웃** — 모바일에서도 잘 보여야 함
2. **앵커 링크** — 각 섹션으로 바로 이동
3. **검색창** — 키워드 검색 기능
4. **주의사항 강조 박스** — 노트/경고 박스 포함
5. **페이지 상단** — "오감리뷰 광고 가이드라인" 표시 + 최종 업데이트 날짜 표시
6. **텍스트 워싱** — 오타 등 교정해서 작성
7. **세부 일정 안내** — 데이터만 입력하면 업데이트 가능한 동적 형태로 구현 (사진 참고)

### 5. 프로(PRO) 설명 페이지
관련 이미지: `프로 설명 페이지/프로 리뷰콘텐츠1.jpg`, `프로 리뷰콘텐츠2.jpg`

**제품 설명**: 오늘의집 내 영향력 있는 크리에이터에게 제품을 제공해 고퀄리티 리뷰 콘텐츠를 획득하는 상품

**기대효과**:
- 콘텐츠 평균 조회수 1.2만회 (1개월/10명 기준, 1회 진행)
- 유저 스타일링샷 평균 접속수 230% 증가
- 고퀄리티 브랜드 에셋 확보
- 크리에이터 SNS 업로드로 노출 효과 증가
- 부가 광고상품 진행 가능

**리뷰 콘텐츠**:
1. 유저들의 스타일링샷
2. 스토어리뷰
3. 유저 개인 인스타그램
4. 콘텐츠 예시 보러가기 (랜딩페이지 링크)

**업로드 횟수** (판매가 기준):
- 200만 원 미만: 1회 (오늘의집 1회 + 인스타그램 1회 + 스토어리뷰 1회 = 총 3개)
- 200만 원 이상: 2회 (오늘의집 2회 + 인스타그램 2회 + 스토어리뷰 1회 = 총 5개)

**단가**:
| 인원수 | 단가(공급가) |
|--------|------------|
| 5명 | 150만 원 |
| 10명 | 250만 원 |
| 15명 | 350만 원 |
| 20명 | 450만 원 |
| 25명 | 550만 원 |
| 30명 | 650만 원 |

**2차 라이선스**: PRO에서 생산된 이미지는 2차 이미지 라이선스 무료 제공
- 활용 범위: 오늘의집 내부(썸네일, 상세페이지, 기획전 등), 자사몰, 자사 SNS
- 타 커머스 및 플랫폼 활용 불가

### 6. 프로 부가광고상품
관련 이미지: `프로 부가광고상품/` 폴더 내 이미지들
(`OGAM LIST.png`, `부가 광고 상품 추천.jpg`, `오감의 아이템.png`, `오늘의집 노출지면 확인방법.jpg`, `오늘의집 인스타그램.png`)

> PRO에 한하여 진행 가능. 오감리뷰 체험단 리뷰 2차 가공을 통해 브랜딩을 위한 추가 노출 콘텐츠 제작 및 노출.
> 부가 광고 상품 콘텐츠는 미션으로 진행되어 15초간 노출, 평균 5만 PV 노출.

#### 상품 1. 오늘의집 인스타그램 — 원브랜드 콘텐츠
- 오감리뷰 체험단 리뷰 2차 가공 → 인스타그램 피드 콘텐츠 제작
- 노출 지면: 오늘의집 인스타그램 피드 1회
- 예상 노출: 팔로워 130만, 평균 11.3만
- 한정 구좌: 월 4개 / 일정 협의 필요
- 단가: **1,000만 원**
- 예시: `https://www.instagram.com/p/DEv_l_CIonv/?img_index=1`

#### 상품 2. 오늘의집 인앱 — 원브랜드 콘텐츠 (오감의 아이템)
- 오감리뷰 체험단 리뷰 2차 가공 → 제품 USP 담은 원브랜드 콘텐츠
- 노출 지면: 오늘의집 인앱 내 오늘의가든 물주기 미션 (1회 고정 노출)
- 예상 노출: 평균 4만~5만
- 한정 구좌: 월 4개 / 일정 협의 필요

- 예시: `https://contents.ohou.se/contents/card_collections/1004579273`

#### 상품 3. OGAM LIST

- 노출 지면: 오늘의집 인앱 내 오늘의가든 물주기 미션 (1회 고정 노출)

## 기타 정책 사항 (사이트에 반드시 포함)

### 오로라 어드민 고객센터 문의 안내
- 오감리뷰 측으로 인입된 크리에이터 문의 → 파트너사와 확인이 필요한 경우 '고객센터 문의' 게시판으로 전달됨
- 오늘의집 고객센터에서 담당자에게 직접 연락 가능
- 고객 문의 처리 지연 방지를 위해 고객센터 연락 시 반드시 응대 필요

### 2차 라이선스 활용 정책
- 제공 콘텐츠: 오감리뷰로 생산된 이미지 & 텍스트 콘텐츠 기준
- 기본 정책: 추가 비용 없이 콘텐츠 활용 허용
- 출처 필수 기재 (미표기 시 법적 이슈 발생)
  예: "오늘의집 000님의 오감리뷰", "오늘의집 000님의 리뷰"
- 타 커머스 및 채널 사용 불가
- 활용 범위: 오늘의집 제품 상품페이지, 파트너사 자사몰/공식 SNS 내 상품 상세페이지·배너
- 자사몰 내 리뷰 페이지 게재는 불가, 그 외 지면 활용 시 오늘의집 사전 협의 필요
- 오프라인 제작물 활용 시 크리에이터 사전 동의 필요
